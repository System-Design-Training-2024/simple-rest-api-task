Flask Product Management API

This is a simple Flask application for managing products in a PostgreSQL database. 
The application provides endpoints to create,retrieve, update, and delete products.

-Prerequisites
Python 3.6+
PostgreSQL
psycopg2 library
python-dotenv library
Flask

-Installation
1.Clone the repository "git clone <repository-url>
cd <repository-directory>"
2.Create and activate a virtual environment "python -m venv venv
source venv/bin/activate  # On Windows use `venv\Scripts\activate`"
3.Install the required packages"pip install -r requirements.txt"
4.Create a .env file in the root directory and add your database configuration " DB_URL = url 
DB_HOST = localhost
DB_NAME = DB_NAME_name
DB_USER = DB_USER_user
DB_PASSWORD = DB_PASSWORD_password
DB_PORT = 5432
"

- Running the Application
1.Load the environment variables "source .env"
2.Start the Flask application"flask run"
3.The application will be available at http://127.0.0.1:5000/
