1. Create Product:
    URL: http://localhost:4000/products
    Method: POST
    Body: 
    {
        "name": "Product 1",
        "description": "This is product 1",
        "price": 100
    }


2. Get All Products:
    URL: http://localhost:4000/products
    Method: GET


3. Update a Product:
    URL: http://localhost:4000/products/id
    Method: PUT
    Body: 
    {
        "name": "New name",
        "description": "New description",
        "price": 200
    }


4. Delete a Product:
    URL: http://localhost:4000/products/id
    Method: DELETE
